Auto PY to EXE

Auto PY to EXE — это графическая оболочка для PyInstaller,
которая значительно упрощает процесс создания .exe файлов из
ваших Python-скриптов. Она позволяет пользователям без опыта работы с
командной строкой легко создавать исполняемые файлы с помощью простого и
понятного интерфейса.
Установка Auto PY to EXE

Для установки Auto PY to EXE выполните следующие команды:
    Установка через pip:

pip install auto-py-to-exe

Запуск приложения: После установки выполните следующую команду,
чтобы запустить графический интерфейс Auto PY to EXE:

    auto-py-to-exe

Как пользоваться Auto PY to EXE

    Выбор файла Python:
        В интерфейсе нажмите кнопку "Browse" рядом с полем "Script Location"
        и выберите ваш .py файл.

    Выбор режима сборки:
        Выберите "One File" (создание одного .exe файла) или "One Directory"
        (создание папки с .exe и всеми зависимостями).

    Дополнительные настройки:
        В разделе "Advanced" можно добавить дополнительные файлы
        или модули, указать иконку для .exe файла, и настроить другие
        параметры сборки.

    Запуск процесса компиляции:
        Нажмите на кнопку "Convert .py to .exe", чтобы начать
        процесс компиляции. Программа выполнит необходимые действия и
        создаст .exe файл в указанной вами папке.

    Тестирование:
        После завершения компиляции тестируйте созданный .exe файл
        на целевой машине, чтобы убедиться, что он работает корректно.

Преимущества использования Auto PY to EXE:

    Простота использования: Не требует знаний командной строки,
    благодаря удобному графическому интерфейсу.
    Гибкость: Позволяет выбирать между созданием одного .exe файла
    или папки с зависимостями.
    Поддержка дополнительных настроек: Возможность добавления файлов,
    изменения иконок, скрытия консольного окна и других параметров.

Auto PY to EXE — отличный выбор для тех, кто хочет упростить процесс
создания исполняемых файлов из Python-скриптов, особенно если вам нужно
создать удобный для пользователей установщик.

=============================================================

Auto PY to EXE позволяет выбирать Python-скрипты для компиляции из
любых директорий на вашем компьютере, независимо от того, где установлен
сам инструмент. Это значит, что вы можете использовать его для
компиляции скриптов из параллельных проектов, находящихся в других папках,
а также из проектов со своими собственными виртуальными окружениями.

При запуске Auto PY to EXE через графический интерфейс,
вы можете указать путь к любому .py файлу на вашем компьютере,
независимо от его местоположения. Инструмент затем выполнит компиляцию в выбранную вами директорию, следуя вашим настройкам.
Работа с виртуальными окружениями

Если ваш проект использует виртуальное окружение, Auto PY to
EXE автоматически подключится к активному окружению, откуда вы запускаете его,
и будет использовать зависимости и интерпретатор из этого окружения для
создания исполняемого файла. Если вы хотите использовать другое
виртуальное окружение, убедитесь, что оно активировано перед запуском
Auto PY to EXE.
Пример:

    Проект A находится в папке C:\Projects\ProjectA, и у него есть
     свое виртуальное окружение.
    Auto PY to EXE установлен в папке C:\PythonTools\auto_py_to_exe.

Вы можете запустить Auto PY to EXE и указать .py
файл из C:\Projects\ProjectA\main.py, чтобы скомпилировать его в
исполняемый файл. Убедитесь, что перед этим вы активировали виртуальное
окружение проекта A, чтобы зависимости корректно подтянулись.

Таким образом, Auto PY to EXE можно использовать с любыми проектами
на вашем компьютере, что делает его гибким инструментом для создания
исполняемых файлов.
